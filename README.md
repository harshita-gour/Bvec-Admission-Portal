# BVEC Admission Portal

A Java-based Admission Management System developed to manage student admission records through a menu-driven console application.

This project was built to practice Object-Oriented Programming (OOP) concepts and understand how real-world admission systems can be structured using Java.

## Features

- Add student admission details
- Store and display student records
- Manage department information
- Manage semester information
- Menu-driven console interface
- Object-oriented project structure

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- VS Code
- Git & GitHub

## How to Run

1. Clone the repository

```bash
git clone https://github.com/harshita-gour/Bvec-Admission-Portal.git
```

2. Open the project in VS Code or any Java IDE.

3. Compile the files

```bash
javac *.java
```

4. Run the application

```bash
java AdmissionPortal
```
